# Dev Tools

Browser utilities for developers and power users — no installation, no account.

- **[JSON Tools](/dev/json-tools/)** — Format, minify, sort keys, stringify, and destringify JSON
- **[List Randomizer](/dev/list-randomizer/)** — Shuffle any list instantly
- **[Random Picker](/dev/random-picker/)** — Pick one or more random items from a list
- **[Sequence Generator](/dev/sequence-generator/)** — Generate numeric sequences with custom prefix, suffix, and separator
